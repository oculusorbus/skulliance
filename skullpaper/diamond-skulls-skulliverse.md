# Skulliverse

![Skulliverse](https://www.skulliance.io/staking/images/screenshots/skulliverse.png)

## Planets

Each core project in the Skulliance is represented with their own planet. Each planet has information on population, percentage of current delegations, and rewards.

## Delegations

The interface tracks current delegations from that project to Diamond Skulls.

## Double CARBON Rewards

If a core project reaches 100% delegation for currently staked Diamond Skulls, the planet will activate and all delegated NFTs for that core project will receive double CARBON rewards.

## Double DIAMOND Rewards

Once all core project planets are activated, the Diamond Skulls planet will activate and all Diamond Skull holders will receive double DIAMOND rewards.
